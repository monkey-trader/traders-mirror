# AddFormCard

`AddFormCard` is a small presentation wrapper for add-forms (New Trade / Add Analysis).

- Purpose: provide a consistent header with title + optional actions and content spacing.
- Responsiveness: mobile-first; content area is stacked and header uses flex with gap.
- Tokens used: `--text`, `--muted`, `--card-border`, `--accent-error`.

Use `AddFormCard` in presentation-level forms to keep layout consistent across features.
